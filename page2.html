Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML needs when i need to read or insert HTML content dynamically, but avoid it for untrusted user input to prevent security risks.
innerText needs when i want to get visible text and ignore hidden elements.
textContent is used when i need all text inside an element, including hidden content, and want better performance.
